== Installation ==

1. Upload the `woocommerce-shareasale-merchant-tracking` folder to the `/wp-content/plugins/` directory
2. Active the WC ShareASale - Merchant plugin through the 'Plugins' menu in WordPress
3. Configure the plugin by going to the `WC ShareASale - Merchant` menu that appears in your admin menu
